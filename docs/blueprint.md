# **App Name**: Aivaz Heritage

## Core Features:

- Legacy Intelligence Insights: A generative AI discovery tool that synthesizes account data to provide personalized wealth strategies and estate blind-spot analysis.
- Dynamic Portfolio Command: Interactive primary dashboard showcasing a high-level overview of diversified assets and performance metrics across the entire family ecosystem.
- Omni-Account Aggregator: A secure portal for tracking and managing multiple individual and trust accounts with real-time balance synchronization using Firestore.
- Wealth Scenario Simulator: A sophisticated AI reasoning tool to model potential future financial outcomes and tax implications based on varying economic conditions.
- Encrypted Strategy Messenger: A private, end-to-end communication hub for secure collaboration between family members and professional advisors.
- Integrated Heritage Strategy: Strategic roadmap builder for managing long-term legacy goals, wealth transition plans, and family values documentation.
- Digital Asset Vault: A military-grade secure file storage repository for essential documents like trusts, wills, and sensitive identification records.

## Style Guidelines:

- The palette is anchored by Obsidian Black (#0B0C10) backgrounds with Silver text (#D4E8F2) for maximum legibility and institutional feel.
- Primary accents use Primary Glass Blue (#4BA3C7) and Deep Glass (#2E5F7A), with Ice Blue (#A8D5E8) for highlighting interactive elements.
- Subtle glassmorphism effects are applied across components using blue tones with transparency and soft highlights to evoke depth.
- Refined pairing of 'Poppins' for headlines and 'Inter' for high-density financial data, ensuring precise and objective legibility.
- Desktop-first layout featuring a fixed 280px left sidebar for primary navigation and a large, fluid-width main content panel.
- Custom, thin-line glassmorphic icons mirroring the Turtle brand logo for a consistent, premium infrastructure aesthetic.
- Swift micro-transitions and lateral slide animations for route changes to maintain a high-performance, responsive user experience.
